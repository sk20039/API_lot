I built an AI-powered API testing tool — and I'd love your feedback.

APIlot is an open-source alternative to Postman with Claude AI built directly into the workflow. Here's what makes it different:

**4 AI-powered features that actually help:**
🔍 Explain — understand what any API response means, field by field
🐛 Debug — diagnose failed requests with specific, actionable fixes
✅ Generate Tests — auto-generate test code in Postman JS, Python, Plain JS, or Shell
💬 Chat — talk to an AI assistant that knows exactly what you just tested

**Built for developers who are tired of:**
- Googling status codes mid-debug
- Pasting JSON into separate AI chats
- Writing boilerplate test assertions by hand

The tech stack: React 18, Zustand, Express, Axios, Claude (via Anthropic SDK), pnpm monorepo.

It's fully open source. Clone it, try it on your APIs, and let me know what you'd change.

👉 https://github.com/sk20039/API_lot
