**Tweet 2 — The problem**

> - Googling status codes
> - Pasting JSON into AI chats
> - Writing tests by hand
> APIlot fixes that.

---

**Tweet 3 — Feature: Explain + Debug**
> Two AI tabs that changed how I debug APIs:
> 🔍 **Explain** — tells you what every field in the response means + flags anything unusual
> 🐛 **Debug** — diagnoses 4xx/5xx errors and gives you 3 specific things to fix
> Both stream in real time. Both have full context of your request + response.

---

**Tweet 4 — Feature: Test Generation**
> ✅ **Generate Tests** — pick your format, click one button:
> - Postman JS (pm.test style)
> - Plain JS (fetch + console.assert)
> - Python (pytest + requests)
> - Shell (bash + curl)
> Copy and paste. Done.

**Tweet 5 — Feature: Chat + Environments**
> 💬 **AI Chat** — free-form conversation with full API context. Ask "why did this fail?" or "how do I add pagination?" and it knows exactly what you were testing.
> Plus: environment variables with {{varName}} substitution, request history, and multiple auth methods.
